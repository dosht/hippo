### S8.5: Silver-Only Read-Path Validation (Quality Gate)

**As Mustafa**, I want to query the silver layer directly via QMD before building extract.py so
that I can confirm compact.py output is rich enough to produce useful gold entries, and stop
early if it is not.

#### Acceptance criteria

- [ ] `silver/` registered as a QMD collection named `hippo-silver`: `qmd collection add ~/src/hippo/silver --name hippo-silver --mask "**/*.md"`
- [ ] The same 5 test questions used in S6 are run against `hippo-silver`; answers are captured for comparison
- [ ] At least one answer contains a concrete specific detail (port number, exact command, file path, or error message), confirming near-miss failures survived compaction
- [ ] At least one no-match query returns empty results (not a hallucinated answer), confirming the collection boundary is correct
- [ ] Human decision recorded in a one-line comment in docs/developer-guide/README.md: "S8.5 go" or "S8.5 blocked: <reason>" before S9 begins
